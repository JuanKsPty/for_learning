La terminal es

Es una interfaz gráfica que simula una linea de comandos y cuando hablamos de una línea de comandos nos referimos a una shell.

  • la Terminal:
    es la ventanita que nos muestra el prompt. Este aloja la shell.
  • Linea de comandos:
    programa que toma comandos y los pasa al sistema operativo para hacer algo.
Un comando es

Un comando puede ser cuatro cosas:

  • Un programa ejecutable.
  • Un comando de utilidad de la `shell`
  • Una función de `shell`
  • Un alias.

El comando type es un comando que nos permite analizar la naturaleza de otros comandos.

Para crear alias en la terminal podemos hacer uso de la palabra reservada alias, este nos crea un alias temporal para la sesión actual en la que estamos trabajando; hacerlo persistente se verá en una siguiente sección.

Wildcards

Son una serie de caracteres especiales que nos permiten realizar mejores búsquedas.

Nos sirven para realizar seccionamiento de archivos o directorios.

Ejemplo “ *.html “ para buscar todos los archivos html.

Redirecciones

Las redirecciones nos permiten realizar comandos extra, a la vez.

Permisos

En la terminal podemos usar diferentes tipos de símbolos, cuales podemos clasificar por:

  • Tipo de archivo

    Atributo Tipo de archivo
    - Un archivo normal.
    d Un directorio.
    l Un link simbolico.
    b Un archivo de bloque especial.

  • Tipo de modo

    Dueño Grupo World
    rwx r-x r-x
    1 1 1 1 0 1 1 0 1
Comandos de basicos

Entre los comandos mas basicos estan:

pwd /* para ver el directorio en el que trabajas. */
ls /* para ver una lista de los elementos del directorio actual. */
mkdir /* para crear un directorio. */
sudo /* para dar permiso de administrador. */
cd /* para cambiar el directorio en el que te encuentras. */
